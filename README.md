# Quick-Osint
an osint tool to show top web-results including target's name and their mail.


requirements:

pip install bs4 && BeautifulSoup

pip install requests && random


INSTALLATION:

```shell
git clone https://github.com/Drac0077/Quick-Osint/
```

```shell
cd Quick-Osint 
```

```shell
chmod +x search.py
```

```shell
./search.py
```
